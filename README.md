# Passion-Pursuit
This project offers users an engaging way to explore diverse STEM careers through the immersive and fast-paced lens of a platformer game. By blending educational themes with classic mechanics, players navigate specialized levels dedicated to Computer Science, Chemistry and Biology, Mathematics, and Engineering.
